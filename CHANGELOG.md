## [2.0.0] - 2026-06-24

### Multi-level curriculum: S1, S2, S3, S4

### Added
- **S1 Physics**: filled 4 gaps (States of Matter, Temperature, Heat Transfer, Expansion). S1 Physics now complete (8 topics).
- **S2 Mathematics**: 9 lessons (Mappings, Vectors, Graphs, Indices/Logarithms, Inequalities, Algebra 2, Similarity, Circle, Set Theory).
- **S2 Biology**: 3 lessons (Soil, Nutrition, Transport).
- **S2 Chemistry**: 3 lessons (Atomic Structure, Periodic Table, Chemical Bonding).
- **S2 Physics**: 3 lessons (Work/Energy/Power, Pressure, Magnetism).
- **S3 Mathematics**: 3 lessons (Trigonometry, Matrices, Ratios/Proportions).
- **S4 Mathematics**: 2 lessons (Composite Functions, Linear Programming).
- **Level selector** on the Learn page (S1, S2, S3, S4 tabs).
- Subject tabs show lesson counts per selected level.
- Graceful empty state when a level+subject has no lessons yet.
